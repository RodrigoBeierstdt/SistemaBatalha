# Sistema de batalha em java
Projeto desenvolvido em Java (POO) com o objetivo de simular um sistema de batalha por turnos entre dois personagens: um Guerreiro e um Orc.
Ações disponíveis:
Atacar
Defender (reduz o dano recebido)
Desviar (chance de evitar o dano)
Inimigo com decisões aleatórias
Controle de vida dos personagens
Verificação de vitória e derrota
